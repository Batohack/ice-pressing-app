## Features

- **Customer Management** (CRUD + Email Alerts)
- **Order System** (Multi-item, Auto-total)
- **Inventory Tracking** (Low Stock Alerts)
- **Payments** (Cash, Mobile Money)
- **Reports Dashboard**
- **Responsive + Animated UI**
- **Bilingual (EN/FR)**
